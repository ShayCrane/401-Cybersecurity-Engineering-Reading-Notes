## What is threat hunting?
***https://www.activecountermeasures.com/what-is-threat-hunting-and-why-is-it-so-important-video-blog/***

**Threat hunting**: 
- Active/proactively search the network for signs of compromise
- especially c2 communications
- not responsive like incident response
- looks to target behaviour, not signatures
- should include all devices on network
- assess by evaluating output
